#2. Ti99/4A

This project mixes 3D printing and electronic tinkering. I wanted to put a raspberry pi 4 inside a Ti99/4A. The aim of the project here is to have a piece of hardware that would allow me to interact with the GPIO pins of the Pi and learn some Python and make it look cool in the process. So you will find below the prints that holds everything together and you will find the electronic part in another section of the site. Enjoy! (stl file #1, stl file #2, stl file #3).
